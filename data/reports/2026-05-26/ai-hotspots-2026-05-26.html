选题 01（P0）： “开源 Agent 工作流周报”怎么变成你的内容护城河？

目标受众
程序员创作者 / 独立开发者 / AI 编程工具重度用户
切题角度
从“新闻转述”升级为“可复刻”：挑 1 个更新点 → 复现 → 给模板与坑位
爆点
开源 + 可复刻 + 可产出（PR/脚手架/模板）= 强收藏
内容结构
一句话变化 → 复现准备 → 跑通小任务 → 迁移到你的项目 → 风险边界
推荐形式
横屏长视频 / 图文长帖（附步骤与 repo 链接）
引用来源
GitHub Agentic Workflows Weekly Update

选题 02（P1）：做“卡顿/冻结/键盘流”的工具体验评测，而不是功能清单

目标受众
效率工具用户 / 工具测评创作者 / AI IDE 用户
切题角度

爆点
CPU 100% / diff 冻结 / 二进制文件卡死，强共鸣
内容结构
复现演示 → 原因解释 → 修复机制 → 10 条 checklist
推荐形式
对比评测短视频 + 配套图文清单
