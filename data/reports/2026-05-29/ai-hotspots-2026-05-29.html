选题 01（P0）｜Claude Opus 4.8：浏览器 Agent 终于能“像人一样做网页任务”了吗？

  • 目标受众：资讯型 AI 受众、效率工具党、做资料/脚本的创作者。
  • 爆点：一镜到底网页任务实测（成功/失败都展示），输出可复用任务清单。
  • 结构：问题背景 → 官方信号 → 3 个任务设计 → 实测 → 复盘 → 风险边界。
  • 形式：屏录视频 + 图文复盘表。
  • 引用：Anthropic Opus 4.8，GitHub Copilot changelog。

选题 02（P0）｜Runway MCP Server：视频生成进入“可编排工作流时代”

  • 目标受众：短视频/广告/教程类创作者、AI 工具控。
  • 爆点：用 3–5 个 MCP 模板演示批量出片、版本管理、自动命名归档。
  • 结构：MCP 30 秒解释 → Runway 能力范围 → 模板演示 → 案例 → 风险兜底。
  • 形式：教程视频 + 模板下载（目录结构与命名规范）。
  • 引用：Runway 官方博客。

  • 结构：信号是什么 → 为什么现在 → 3 个应用场景 → 风险清单 → 创作者策略。
